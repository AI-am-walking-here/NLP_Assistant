## Collection

### Source corpora (external — not redistributed raw)

We **do not** redistribute the original unarXive shards or arXiv S3 LaTeX tarballs. Those are obtained from:

| Source | Link | Our use |
|--------|------|---------|
| **unarXive** (2016–2022 LaTeX JSON) | [Zenodo 7752615](https://zenodo.org/record/7752615) | ~90% of papers |
| **arXiv S3** (2023–2025 LaTeX) | [arXiv bulk data](https://arxiv.org/help/bulk_data) | ~10% of papers |
| **arXiv metadata** (filtering) | [Kaggle arXiv snapshot](https://www.kaggle.com/datasets/Cornell-University/arxiv) | cs.CL ID filter |

Re-download from scratch: enable sources in `configs/data.yaml` and run `scripts/s3_pull.py` (see `docs/DATA_ACQUISITION.md`).

### What we include

1. **`papers.jsonl.gz`** (external archive) — processed snapshot after LaTeX normalization.

2. **Split lists** — Deterministic stratified splits (`random_seed: 1337`): index (7,901), eval holdout (878), eval grid (80), SFT (2,174), graph pilot (500).

3. **SFT / RankRAG JSONL** — Generated by `scripts/build_sft_data.py` and `scripts/build_rankrag_data.py` from parsed papers + retrieval index. Each SFT row: `(title, outline) → target abstract` with optional retrieval-conditioned prompts.

4. **Eval set** — 80 prompts built by `scripts/build_eval_prompts.py` from holdout papers (title + contribution outline as input; human abstract available for audit, not used at generation time).

## Known biases and limitations

- **Category:** Primary arXiv category **cs.CL** only — not representative of all NLP or all arXiv.
- **Language:** English scientific prose; LaTeX parsing failures excluded (~2% failed/partial).
- **Time window:** 2016–2025 mix (unarXive-heavy 2016–2022, S3 LaTeX 2023–2025).
- **Length filter:** Papers with body text &lt; 4,000 characters dropped (~2% of parsed).
- **SFT selection:** SFT split prefers papers with ≥ 3 citation keys (`sft_citation_min: 3` in config at build time).
- **Eval grid:** 80 prompts — small for statistical power; FActScore scores are verifier-dependent (70B Llama).
- **Retrieval bias:** FAISS + BGE-large favors lexical/semantic similarity to outline; Graph RAG pilot covers 500 papers only.
- **Generator bias:** Llama-3.1-8B + domain SFT inherits model and training-data biases.
